## Списки

Чтобы добавить ненумерованные списки, необходимо пунктры выделить звёздочкой. Например, вот так 

* Элемент 1
* Элемент 2
* Элемент 3

Чтобы добавить нумерованные списки, необходимо просто пронумеровать пункты. Например, вот так:

1. Первы йпункт
2. Второй пункт
3. Третий пункт
